# vencord plugins

My public vencord userplugins. One folder per plugin under `userplugins/`.

Not official. Not affiliated with discord or vencord.

Right now only dm last seen is here. Might add more later.

## DM-last-seen

Vencord userplugin that shows online, idle, or last seen next to DM names. Gray text like `on-5m`, `idle-2h`, or `last seen ?` when the time is unknown.
